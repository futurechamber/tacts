- Overlapping windows
- Multivariate TS support
- Sampling Rate normalization
- Non-biased Unit Cost fitting
- Support for Gap Masking
- Parallel Lambda Optimization
- Parallel Spectrum Pipeline
- Surrogate Bootsrapping
